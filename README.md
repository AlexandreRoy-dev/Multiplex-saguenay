# Multiplex Saguenay

Site statique pour René Bouchard — vente de multiplex et immeubles à revenus au Saguenay–Lac-Saint-Jean.

## Pages SEO

12 pages de destination (`*.html` à la racine) ciblent mots-clés locaux : duplex, triplex, quadriplex, Chicoutimi, Jonquière, La Baie, Alma, Lac-Saint-Jean, courtier, évaluation, etc.

Regénérer les pages après modification du script :

```bash
node scripts/generate-seo-pages.js
```

## Fichiers

- `index.html` — page principale
- `sitemap.xml` / `robots.txt` — référencement
- Mettre à jour le domaine `multiplexsaguenay.com` dans le générateur si l’URL de production diffère.
